    handle_cast(msg, state)

    View Source

    GenServer.cast/2 callback.

    • :push_item Insert a {list, item_id} tuple into state.
    • :sync_list Fetch list item IDs from HTTP endpoint and keep only the first @sync_items_count entries. Save the entries to a temporary upcoming EslNews.Store.List, and then create empty stub EslNews.Store.Story for each ID before pushing the ID into the processing queue.
    • :sweep_list Fetch list item IDs from an upcoming list and hydrate them into EslNews.Store.Story structs. Ensure no stub structs remain by checking their type attribute and once all are set, save the upcoming list as the current list. Finally, delete the upcoming list to prepare for the next sync cycle.
