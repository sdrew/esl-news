     Modules

    EslNews

    EslNews works as an aggeregator for Hacker News stories, fetching data from the HN API for further analysis or consumption.

    EslNews.Application

    EslNews.Application sets up a :cowboy HTTP webserver, the main Supervisor for the EslNews.Http.SyncLists and EslNews.Http.SyncItems workers, and for EslNews.Store.Schema which handles the :mnesia storage.

    EslNews.Handler

    EslNews.Handler provides a standard base implementing the :cowboy_rest behaviour and URI param extraction for EslNews.Handlers

    EslNews.Handlers.Stories

    EslNews.Handlers.Stories provides a JSON response for the latest available Stories. Implements @behaviour for EslNews.Handler

    EslNews.Handlers.Story

    EslNews.Handlers.Story provides a JSON response for a single Story resource. Implements @behaviour for EslNews.Handler

    EslNews.Handlers.Ws

    EslNews.Handlers.Ws provides a JSON response for the latest available Stories through websockets. Implements @behaviour for EslNews.Websocket

    EslNews.Http.Client

    EslNews.Http.Client fetches JSON objects from an HTTP API.

    EslNews.Http.MockClient

    EslNews.Http.MockClient mimicks EslNews.Http.Clientbut loads its JSON objects from disk, instead of the network.

    EslNews.Logger

    EslNews.Logger provides logging helpers for EslNews.Handlers

    EslNews.Router

    EslNews.Router defines the routes and handlers required to generate the :cowboy_router.dispatch_rules() required by the :cowboy webserver.

    EslNews.Store.List

    EslNews.Store.List provides a :mnesia schema and struct to store EslNews.Store.List objects, as well as helper methods to interact with the storage table.

    EslNews.Store.Schema

    EslNews.Store.Schema sets up the :mnesia application and creates tables for EslNews.Store.List and EslNews.Store.Story.

    EslNews.Store.Story

    EslNews.Store.Story provides a mnesia schema and struct to store EslNews.Store.Story objects, as well as helper methods to interact with the storage table.

    EslNews.Websocket

    EslNews.Websocket provides a standard base implementing the :cowboy_websocket behaviour and URI param extraction for EslNews.Handlers
